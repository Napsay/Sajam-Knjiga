# 📚 SajamKnjiga

> Predmetni projekat iz predmeta **Osnovi informacionih sistema i softverskog inženjerstva**  
> Fakultet tehničkih nauka, Univerzitet u Novom Sadu

---

## 📖 Opis projekta

**SajamKnjiga** je desktop aplikacija namenjena upravljanju sajmom knjiga. Aplikacija korisnicima omogućava kreiranje, pregled, izmenu i brisanje podataka (CRUD operacije) vezanih za knjige, autore, izdavače i izlagače na sajmu.

Projekat je razvijen kao deo predmetnih obaveza na predmetu *Osnovi informacionih sistema i softverskog inženjerstva (OISISI)*.
